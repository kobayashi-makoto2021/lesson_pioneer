# CLAUDE.md — lesson_pioneer

## プロジェクト概要

子供向けプログラミング教室「アイアルク」パイオニアコースの教材リポジトリ。
対象：小学4年生〜中学1年生（プログラミング歴1年以上、3年目相当）。
全48レッスン、1回60分。

## カリキュラム構成

- **フェーズ1（レッスン01〜16）**: toio Do によるアルゴリズム学習
- **フェーズ2（レッスン17〜32）**: Python テキストコーディング
- **フェーズ3（レッスン33〜48）**: Arduino Uno による電子工作

詳細は README.md を参照。

## ファイル命名規則

- 生徒向けテキスト: `レッスン_xx.md`（例: `レッスン_01.md`）
- 講師向けテキスト: `講師用テキスト.md`
- 画像フォルダ: `image/`
- フォルダ名: `lesson_01/` 〜 `lesson_48/`（ゼロ埋め2桁）

## 執筆方針

- **対象年齢を意識する**: XY座標・変数など「当たり前」と思える概念も丁寧に導入する
- **Scratch→Pythonの翻訳**: フェーズ2では「Scratchではこう書いていた→Pythonではこう書く」形式で説明する
- **写経中心**: Pythonフェーズはタイピング・インデント習得を重視し、アクティビティは深く考えなくてよい
- **概念の依存関係を守る**: 例）二重ループは全探索の前、スタック/キューはソートの前
- **toioロボット**: フェーズ1では2台使用可能（ロボット1: 宝を埋める役、ロボット2: 探索役）

## カリキュラム変更時の注意

- レッスン項目の削除・統合・移動は必ずユーザーに提案・確認してから実行する
- 学習順序には教育的な依存関係があるため、変更時は前後のレッスンへの影響を確認する
