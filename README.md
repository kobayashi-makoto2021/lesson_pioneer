# lesson_pioneer

子供向けプログラミング教室「パイオニア」コースのレッスン教材リポジトリです。

## 概要

| 項目 | 内容 |
|---|---|
| 対象 | 小学4年生〜中学1年生（プログラミング歴1年以上） |
| レッスン数 | 全48レッスン |
| レッスン時間 | 1回60分 |
| 運営 | アイアルク（受講3年目相当） |

## カリキュラム構成

### フェーズ1：初期 — toio Do によるアルゴリズム学習（レッスン01〜16）

Sony製ロボット教材「toio」と、Scratch風ビジュアルプログラミング環境「toio Do」を使用。
ロボットを動かしながらアルゴリズムの基礎概念を身につける。

| 02 | だるまさんがころんだ！ — フラグで制御しよう | ボタンを押している間だけtoioを動かす | フラグ変数（真偽値）／フラグによるループ制御 |

## 生徒の提出ファイル命名規則

生徒が保存するプログラムファイル・スクリーンショットのファイル名は以下のルールで統一する。

**形式：** `日付（4桁）名前（ローマ字）番号（2桁）.拡張子`

| 項目 | ルール | 例 |
|---|---|---|
| 日付 | 月日の4桁（ゼロ埋め） | 5月10日 → `0510` |
| 名前 | ローマ字（名のみ、小文字） | まこと → `makoto` |
| 番号 | そのレッスンで保存する順番（2桁） | 1番目 → `01` |

**例：** 5月10日・まこと・1番目のファイル → `0510makoto01.sb3` / `0510makoto01.png`
